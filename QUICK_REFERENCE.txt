TO CHANGE ADMIN PASSWORD
========================
Edit: src/admin/AdminApp.tsx
Line: ~45
Find: if (password === 'admin123')
Change 'admin123' to your password


TO ADD NEW MENU CATEGORY
========================
1. Edit: src/types/index.ts
   Add to Category type

2. Edit: src/components/CategoryFilter.tsx
   Add new tab button

3. Edit: src/admin/AdminApp.tsx
   Add option to category select


TROUBLESHOOTING
===============
Data not syncing    ‚Üí Set up Firebase
UPI not updating    ‚Üí Check storeConfig.upiId
Location error      ‚Üí Enable HTTPS
No alarm sound      ‚Üí Click anywhere on page first
Theme stuck         ‚Üí Clear localStorage


RESET ALL DATA
==============
Open browser console:
> localStorage.clear()
> location.reload()
